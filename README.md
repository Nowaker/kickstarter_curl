# KickstarterCurl

A simple command line tool that collects metadata from a Kickstarter project page.

## Installation

Add this line to your application's Gemfile:

    gem 'kickstarter_curl'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install kickstarter_curl
